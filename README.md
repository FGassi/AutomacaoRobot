Este projeto visa colocar em prática o treinamento relacionado ao framework Robot. O framework Robot é utilizado como ferramenta de automação para validação de cenários. Estes, que por sua vez, foram elaborados através da linguagem Gherkin, muito comumente usada para descrição de cenários de testes de comportamento.
Para o projeto em questão, foi utilizado o site https://practice.expandtesting.com/
