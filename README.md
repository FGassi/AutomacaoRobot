Este projeto tem como objetivo demonstrar o uso do Robot Framework como ferramenta de automação de testes.
Os cenários foram elaborados utilizando a linguagem Gherkin, amplamente adotada para a descrição de cenários de comportamento (BDD).

Para a execução dos testes, foi utilizado o site de prática: https://practice.expandtesting.com/

This project aims to demonstrate the use of the Robot Framework as a test automation tool.
The test scenarios were created using the Gherkin language, widely adopted for describing behavior-driven development (BDD) scenarios.

The tests were executed using the practice website: https://practice.expandtesting.com/
