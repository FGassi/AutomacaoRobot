Este projeto visa colocar em prática o treinamento relacionado ao Robot framework. O framework é utilizado como ferramenta de automação para validação de cenários. Estes, que por sua vez, foram elaborados através da linguagem Gherkin, muito comumente usada para descrição de cenários de testes de comportamento.
Para o projeto em questão, foi utilizado o site https://practice.expandtesting.com/

-

This project aims to put into practice the training related to the Robot Framework. The framework is used as an automation tool for scenario validation. These scenarios, in turn, were created using the Gherkin language, which is commonly used to describe behavior-driven test scenarios. For this project, the website https://practice.expandtesting.com/
 was used.
